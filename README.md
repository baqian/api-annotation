# api-annotation
like java annotation, using comments annotation,  generate both router binding script and apidoc

## Syntax

```
/**
 * @api {get|post|PATCH|$method} /api/$version/user/:id
 * @name User.getUser
 *
 * @param
 *    - id
 *
 * @query
 *    -
 *    --
 *
 * @body {}
 *
 * @jsonbody
 */
```
